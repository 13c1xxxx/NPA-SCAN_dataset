数据集：在智能合约漏洞检测领域目前还有公认的基准数据集，本文通过整合DAppSCAN-source、SmartBugs wild dataset以及自建数据集构建了一个包含47140个真实世界的智能合约的多模态数据集，除删除注释、多余空格、缩进外，还使用了minHash算法和合约地址对比的方式对solidity文本内容进行去重，保证了数据集的质量。将整个数据集按照80%、10%、10%的比例随机分为训练集、验证集和测试集。
此外，在对比实验中使用了PSCVFinder[18]数据集，其中包括208726条智能合约漏洞数据，涵盖了可重入漏洞和时间戳依赖漏洞，本文在其源码基础上扩充了自然语言描述和抽象语法树两种模态。
